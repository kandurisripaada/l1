## 🌟 Overview

**LanguaBot** isn't just another language app; it's your **intelligent conversation partner**. Built with cutting-edge AI, it simulates real-world interactions to help you gain fluency, confidence, and mastery over the English language.

Whether you're prepping for a high-stakes job interview or just want to chat, LanguaBot adapts to you.

## ✨ Key Features

### 🤖 AI Interviewer
Step into the hot seat with our realistic AI Interviewer. Choose your challenge:
- **👔 HR Round**: Master behavioral questions and cultural fit assessments.
- **💻 Technical Round**: Deep-dive into coding concepts and system design.
- **🧠 Behavioral & Mixed**: A comprehensive test of your soft and hard skills.

### 📚 Adaptive Grammar Gym
Forget boring textbooks. Our AI generates **unlimited, unique grammar exercises** tailored specifically to your level:
- **🟢 Basic**: Essential structures for beginners.
- **🟡 Intermediate**: Refine your sentence construction.
- **🔴 Advanced**: Master complex nuances and professional phrasing.

### 🎧 Immersive Listening
Enhance your ear for detail with dynamic audio content:
- **🗣️ Conversations**: Realistic dialogues with varying accents and speeds.
- **📖 Stories**: Engaging narratives followed by comprehension checks.

### 🎙️  Voice-Powered Interaction
Don't just type—**speak!** Integrated Speech-to-Text allows for fluid, hands-free practice, making your learning experience natural and effective.

## ⚡ Quick Start

### Prerequisites
- **Node.js** (v14+)
- **MongoDB** (Local or Atlas)

### 🚀 Backend Setup

1.  **Clone & Install**
    ```bash
    cd backend
    npm install
    ```

2.  **Configure Environment**
    Create a `.env` file in the `backend` folder:
    ```env
    PORT=5000
    MONGODB_URI=mongodb://localhost:27017/languabot
    JWT_SECRET=super_secret_key_change_this
    GOOGLE_CLIENT_ID=your_google_client_id
    GROQ_API_KEY=your_groq_api_key
    ```

3.  **Ignition**
    ```bash
    npm run dev
    ```

### 🎨 Frontend Setup

1.  **Install Dependencies**
    ```bash
    cd frontend
    npm install
    ```

2.  **Launch**
    ```bash
    npm start
    ```
    Visit `http://localhost:3000` and start your journey!
